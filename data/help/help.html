Cosmographia Help

Basic navigation

Mouse

  • Left drag orbits center object
  • Right drag pans the view
  • Mouse wheel changes distance to center object
  • Single click selects an object
  • Double click an object to visit it
  • Shift+right drag to change field of view

Keyboard

  • Arrow keys pan the view

Gestures

  • On supported trackpads, pinch or expand to change the field of view


For more information, visit the Cosmographia website.

Time

The current time may be set by clicking on the time display in the upper right corner. The time rate is controlled with the following keys:
  • Space - Pause/unpause time
  • ⌘L - Increase time rate 10x
  • ⌘K - Reduce time rate 10x
  • ⌘J - Reverse time direction
