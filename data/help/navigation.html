Navigation

Movement in Cosmographia is relative to the current central object. The central object is indicated by a rotating marker or by an arrow when it is offscreen.

  • Left drag the mouse to rotate around the central object
  • Right drag to pan the view without changing position
  • Use the mouse wheel or trackpad scrolling to change the distance to the central object
  • Double click on an object to travel close to it

The orientation of the view can be adjusted by pressing the arrow keys. On computers with a trackpad, you may use the rotate gesture to roll the view about the center point.

The field of view can be adjusted in two ways:

  • Shift + right dragging the mouse
  • The pinch gesture on computers with trackpads
The field of view is shown at the top of the screen while it is being adjusted.

Some navigation commands are available via the menu that appears when right clicking an object:

  • Set as center makes the clicked object the new central object
  • Set as fixed center makes the clicked object central and keeps the view position fixed with respect to the center.
  • Track is used to follow the central object while keeping the tracked object in view.
