Cosmographia was developed by Chris Laurel, but it relies on open source software libraries from many other authors.

VESTA visualization library
ASTOS Solutions Gmbh

Qt4 cross-platform framework
Nokia Corporation

Eigen C++ templated-based matrix library
Benoît Jacob and Gaël Guennebaud, plus many other contributors

lib3ds, a library for loading Autodesk 3D Studio files
Jan Eric Kyprianidis

GLEW, the OpenGL Extension Wrangler
Milan Ikits, Marcelo Magallon and other contributors

QJson, a Qt4 library for reading/writing JSON
Flavio Castelli

The code to compute the positions of the satellites of Mars, Jupiter, Saturn and Uranus is adapted from work by Johannes Gadjosik, who translated FORTRAN code originally produced at IMCCE into C.
